# 个人简历网站

这是一个纯静态的单页简历网站，可以直接用浏览器打开 `index.html` 预览。

## 下一步替换内容

- `index.html`：替换姓名、职位定位、经历、项目、技能、教育和联系方式。
- `contact@yinyq.net`：当前个人域名联系邮箱。
- 头像区域：可以把占位视觉替换成你的个人照片。

## 后续部署方向

- 最简单：GitHub Pages
- 自定义域名和性能更好：Cloudflare Pages
- 前端项目扩展空间更大：Vercel

## 域名邮箱方向

如果你想继续使用 Gmail 收发个人域名邮箱，常见路线有两种：

- Google Workspace：最省心，域名邮箱原生就是 Gmail。
- 邮件转发 + SMTP：收件转发到 Gmail，发件在 Gmail 里添加“发送邮件地址”，需要额外 SMTP 服务和 DNS 记录。

详细推进清单见 `EMAIL_SETUP.md`。
